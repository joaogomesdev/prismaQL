## 💻 Project
  🟣 A GraphQL API to practice PRISMA concepts implemented in Node ⌛

## 🔨 Features

- [X] Create user
- [X] List users

## ✨ Technologies

- [X] Typescript
- [X] ts-node-dev
- [X] GraphQL
- [X] Prisma


## 🏃‍♂️ Running the project

Clone the repository

```bash 
  git clone https://github.com/joaogomesdev/prismaQL.git
  cd prisma
```
Install the dependencies

```bash
  yarn 
```

Run the project 

```bash
  yarn dev
```
